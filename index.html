Welcome

My name is Jon Chan, and I'm the creator of Bento. I made Bento to help people learn how to code. As someone that's completely self-taught in web development, I learned to code on the web when there were few resources out there. Things are different now - there are so many ways to learn that it's hard to figure out where to start. I'll be your guide in navigating the many resources out there. This is my way of passing on the knowledge that I have about coding to others. Let's get started
